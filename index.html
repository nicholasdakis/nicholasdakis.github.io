Level Up!
BETA
Gamified health and fitness app
Web (all devices) PWA (installable, native-like experience) Android (coming soon)
Features
  • Log meals via barcode scan, search, or manual entry with full macro breakdown and serving size adjustment
  • Set calorie and macro goals with live progress tracking in the food logging tab
  • Analytics tab for reviewing daily or range-based calorie and macro intake
  • Calorie calculator (Harris-Benedict, Mifflin-St Jeor) with BMR and TDEE breakdowns in imperial and metric units
  • Check in to nearby locations on an interactive map to earn XP with 24-hour per-location cooldowns
  • Badges, global leaderboard, daily XP rewards, streak tracking, and push notifications
  • Google and email sign-in with cross-device sync for username, theme color, and profile picture
  • Responsive UI with adaptive layouts across mobile, tablet, and desktop screens
Under the hood
  • Flask backend organized into auth, service, and repository layers with shared authentication middleware across routes
  • PostgreSQL schema optimized for atomic XP and level updates using row-level locking for concurrent safety
  • Redis caching and distributed locking to reduce external API calls and prevent cache stampedes under load
  • Token-bucket rate limiter backed by PostgreSQL with daily refills and failure-safe quota rollback
  • Background jobs integrated with Firebase Cloud Messaging for push notifications and automatic invalid token cleanup
  • CI/CD pipeline via GitHub Actions for automated testing and deployment
